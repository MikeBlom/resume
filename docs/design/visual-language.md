# Visual Language & Originality Guidelines

This project is inspired by the *feeling* of NES-era platformers. It must never read as a
re-skin of any Nintendo property. Copyright protects specific sprites, music, and level designs;
**trade dress** can protect a combined look. These rules keep the design original — and are
themselves part of the public case study.

## Hard prohibitions

- No question-mark blocks, warp pipes, mushrooms/power-up flora, coins with the classic chime,
  flagpoles, castles-as-level-ends, koopa/goomba-adjacent creatures, or brick-smashing.
- No SMB palette signature (that sky-blue + brick-brown + green-pipe combination).
- No character silhouette resembling Mario/Luigi (no red cap + overalls + mustache ensemble).
- No recreations or sound-alikes of specific Nintendo jingles or SFX.
- No copied level layouts, including the iconic 1-1 opening rhythm.

## What we build instead

- **Palette:** derived from the story — Dutch brick reds/canal greens/tulip fields (ch. 1),
  muted earth tones (ch. 2), cool blues/terminal greens (ch. 3), dawn gradient (ch. 4). Every
  text/background pairing contrast-checked (WCAG AA minimum).
- **Character:** original pixel interpretation of Mike at 3–4 ages, shared rig with
  palette/accessory swaps; distinctive silhouette (to be designed from photo references).
- **Interactive objects:** story-native — newspaper bundles, bicycles, duffel bags, radios,
  laptops, whiteboards — never abstract "reward blocks."
- **Transitions:** original set pieces per chapter boundary (e.g., harbor/airport departure,
  a compiling gate) — designed in M2, documented here.
- **Audio:** CC0 or original chiptune-adjacent tracks, muted by default; attribution in
  `public/assets/ATTRIBUTION.md`.
- **Typography:** open-licensed pixel font (license verified before inclusion).

Every asset added to `public/assets/` gets a line in `ATTRIBUTION.md` with source and license.
