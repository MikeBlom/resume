# REQ-001 — Milestones & POC Scope

Sequenced for an active job search: shareable value ships at every gate.

## M0 — Foundation + shareable traditional resume (week 1)

Public repo with transparency skeleton, CI pipeline, Cloudflare Pages deployment on custom
domain, typed content model holding the real resume, and a polished printable `/resume` page.
**The URL is usable on job applications after M0.**

## M1 — Proof of concept world (weeks 2–3)

- Phaser 3 world inside the React shell; placeholder character (one age), walk + jump.
- Keyboard controls and tuned touch controls — full mobile parity.
- ~2 screens Netherlands environment → one original transition set piece → ~2 screens tech
  environment.
- 4–6 interactive resume objects rendered from the content model.
- Mini-game **shell** only: a doorway → placeholder scene → return to world position.
- Cloud layer reachable at any time (press/swipe up): navigation + resume.
- Reduced-motion page mode; audio toggle with one ambient CC0 track.

**POC gate — exit questions:** Delightful or gimmick? Info legible in <60s? Accessible path
equivalent? Mobile actually good? Does the architecture scale to four chapters? Engine verdict
(keep Phaser or swap for PixiJS / DOM world).

## M2 — Chapters 1 & 3 (resume-critical)

Real art direction + visual-language doc, aging character looks, full Netherlands and
edtech/leadership chapters, deep links, all four review gates run and committed.

## M3 — Chapter 2 (military) + paper-route mini-game

Respectful military chapter gated on Mike's public-comfort sign-off list; 3-lane paper-delivery
mini-game; aircraft-coordination game lands here or M4 by capacity.

## M4 — Chapter 4 + remaining games + case-study completion

Unwritten-future chapter with call to action, engineering-leadership mini-game (concept chosen
at M3 gate), on-site transparency hub, published cost analysis and retrospective, launch polish.

## Out of scope (v1)

Multiplayer/social features, CMS/admin UI, server-side anything (fully static), visual
regression testing (post-v1), commissioned artwork/music beyond CC0 + original pixel work.

Every milestone ends with: review reports committed, cost ledger updated, prompt log updated,
deployed.
